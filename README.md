# QuirkyCoders Readme

## Background

This document was created to aid in getting started with frontend development on the Quirkycoders Project.

## Getting Started

### For local development, set the following environment vars:

```
REACT_APP_API_URL=http://localhost:3001/api/v1
REACT_APP_API_TOKEN=8a935a34838acaa35d9f48bd0e21a51a
```

### Installing

#### Clone Repository

`git clone <this repo>`

#### Install dependencies

`yarn`

#### Start Application

`yarn start`

#### Deploy Application

Pushing to development / production will automatically deploy updates to staging / prod respectively.

`git push origin [develop/master]`
